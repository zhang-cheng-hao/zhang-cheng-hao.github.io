实习经历
推荐系统算法实习：LLM 常识对齐的 Item 表征
推荐召回 / Item Representation / LLM Alignment / 优秀 LR

现有推荐系统中的物品表征对齐主要依赖推荐/搜索域共现信号与多模态信号。前者擅长捕捉高热行为关系， 例如牙膏与牙刷的高频同购，但长尾覆盖不足且易受行为噪声干扰；后者能够识别图文相似商品， 但容易把牙膏与药膏这类外观相似但使用场景不同的商品拉近。二者共同缺口是缺少与真实世界常识的语义对齐。

我们没有直接把 LLM item embedding 作为精排特征，而是让 LLM 判别用户历史序列中的 item 相关性， 构造对比学习样本，将 LLM 表征对齐到推荐域分布；随后在 SIM GSU 中使用 LLM embedding 进行 soft search，挖掘与目标商品语义相关、但未被共现关系覆盖的用户长期兴趣序列。

结果：离线训练 AUC 提升 0.45pp，内循环大盘总全局后验预期花费提升 1.883%。
